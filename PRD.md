# PRD: 愤怒的猪 3D

> **v3.0（2026-07-17）范围扩大**：关卡规模 3 关 → **100+ 关（5 张主题地图 × 20 关）**；新增 **TNT 炸药箱**；新增 5 套地图主题（草原/沙漠/雪山/火山/天空城）；全部第三方库 vendored 本地化（去 CDN 依赖）；建立关卡生产管线（生成器+无头物理验证+自动求解器）；补充防侵权合规要求（原创美术/音效、禁用词扫描、原创声明）。以下 v2.1 玩法数值体系继续有效，除非 v3.x 明确覆盖。

## Problem
做一个"角色互换"版的愤怒的小鸟——猪用弹弓发射自己去砸鸟。3D 立体版本，手机/电脑浏览器均可游玩。要还原当年玩愤怒的小鸟的感觉。部署在 angry-pigs 仓库（GitHub Pages），可分享给好友。

## Scope
**In:**
- 3D 立体游戏，Three.js 渲染 + cannon-es 物理引擎
- 横屏布局，手机/桌面浏览器均可
- 3 关，难度逐渐递增，验证可玩性
- 核心玩法：拖拽弹弓发射猪，砸坏结构物，击中鸟
- 三种猪：普通猪（粉，无技能）、飞速猪（黄，点击加速）、炸弹猪（黑，点击爆炸）
- 两种鸟状态：飞鸟（固定圆心范围随机飘移）、站鸟（平台宽度 1/4 范围左右跑动）
- 鸟与建筑物无碰撞但不穿过；鸟被击中后掉落遵循真实物理
- 三种建筑材料：冰（最脆）、木（中等）、石（最硬）
- 默认固定斜视角相机，带相机锁定/解锁按钮，解锁后可手动旋转+缩放
- 代码精细建模（几何体组合），圆润卡通风格，简单待机动画
- 音效：程序化生成卡通风格音效，风格接近原版
- 分数 + 三星评级系统
- localStorage 本地存档
- 手机竖屏时显示横屏提示

**Out:**
- 云端存档（后续升级）
- 自定义域名（后续考虑）
- 10 关扩展（验证后再做）
- 鸟反击机制
- 多人对战
- 排行榜

## Non-goals
- 外部 3D 模型文件（用代码建模）
- 骨骼动画/毛发纹理（超出代码建模能力）
- 真实愤怒的小鸟音效文件（版权问题）
- 后端服务器（第一版纯前端）

## Requirements

### R1: 3D 物理引擎
1. Three.js 渲染 3D 场景，cannon-es 处理物理模拟
2. 重力、碰撞、反弹、摩擦力物理模拟
3. 猪被发射后受 3D 物理规则控制（3D 抛物线）
4. 猪撞击结构物和鸟时产生物理碰撞
5. 结构物受撞击后按材料耐久度损坏（冰1次、木2-3次、石多次/炸弹猪）
6. 猪落地后停止，可重新发射下一只猪

### R2: 猪系统
1. 三种猪：普通猪（粉，无技能）、飞速猪（黄，点击加速）、炸弹猪（黑，点击爆炸）
2. 每关按预设顺序发射猪
3. 弹弓上显示下一只要发射的猪
4. 屏幕左下角显示剩余猪队列
5. 飞行中点击屏幕触发特殊技能（飞速猪加速 / 炸弹猪爆炸）
6. 猪有简单待机动画（在弹弓上轻微晃动）

### R3: 鸟系统
1. 飞鸟：在固定圆心范围内随机飘移飞行，碰边界转向
2. 站鸟：在平台宽度 1/4 范围内左右来回跑动，到边缘转身
3. 飞鸟飘移轨迹避开建筑物——不穿过也不碰撞，碰到建筑物边界转向
4. 鸟被猪击中后切换为掉落状态：受重力影响，与建筑物和地面产生碰撞
5. 鸟掉落后落到地面或建筑物上，停留后消失
6. 鸟有简单待机动画（飘移时翅膀微动 / 跑动时轻微弹跳）

### R4: 建筑材料
1. 冰块：半透明蓝色，最脆，1次重击碎裂，轻撞会滑
2. 木箱：棕色木纹，中等，2-3次重击碎裂
3. 石块：灰色粗糙，最硬，需要炸弹猪或多次撞击才能破坏
4. 结构物被破坏时产生碎片效果

### R5: 弹弓发射
1. 鼠标/手指按住猪往后拖拽
2. 拖拽时显示弹弓拉伸视觉效果 + 力度指示器
3. 松手发射，猪按拖拽反方向飞出
4. 拉得越远力度越大
5. 拖拽 X 轴映射左右方向，Y 轴映射前后（力度），自动带向上抛物线弧度
6. 发射后等猪停止/消失，下一只自动装填

### R6: 相机系统
1. 默认固定斜俯视角，看着整个关卡
2. 屏幕上有相机锁定/解锁按钮，点击切换
3. 按钮状态明确显示当前是锁定还是解锁
4. 解锁状态：桌面鼠标右键拖拽旋转，滚轮缩放；手机单指拖拽旋转，双指捏合缩放
5. 锁定状态：不响应旋转缩放操作
6. 旋转角度限制在合理范围（不能转到地面以下）
7. 缩放限制在最小/最大距离之间

### R7: 关卡系统
1. 3 关，难度递增
2. 第1关：教学关，1只鸟，简单结构，3只普通猪
3. 第2关：进阶，2只鸟（飞鸟+站鸟），多层结构，含飞速猪
4. 第3关：挑战，3只鸟，复杂结构，含炸弹猪和石块
5. 关卡选择界面显示已通关关卡的星级
6. 第一关通关后解锁第二关，以此类推
7. 最后一关通关后显示"恭喜全部通关"

### R8: 分数与星级
1. 每击落一只鸟 +1000 分
2. 每剩余一只猪 +500 分
3. 星级：一星=通关，二星=达到目标分数线，三星=更高目标分数线
4. 每关有 twoStar 和 threeStar 分数线

### R9: 存档系统
1. localStorage 自动保存：最高解锁关卡、每关最高分、每关最高星级
2. 关卡选择界面显示已通关关卡的星级
3. 刷新页面不丢进度

### R10: 美术与建模
1. Three.js 几何体组合精细建模，圆润卡通风格
2. 猪：圆滑球体身体 + 鼻子 + 耳朵 + 眼睛 + 尾巴，颜色区分种类
3. 鸟：圆滑球体身体 + 嘴 + 翅膀 + 眼睛
4. 冰块：半透明蓝色立方体；木箱：棕色木纹立方体；石块：灰色粗糙立方体
5. 弹弓：Y形木叉
6. 地面：草地平面；天空：渐变蓝色天空盒
7. 平滑着色 + 简单光照
8. 角色待机动画

### R11: 音效
1. 发射音效（弹弓拉伸+释放）
2. 撞击音效（冰脆、木闷、石沉，不同材质不同声音）
3. 猪叫、鸟叫
4. 通关/失败音效
5. 程序化生成（Web Audio API），卡通风格接近原版但不完全一样

### R12: UI 与交互
1. 横屏布局
2. HUD：分数、剩余猪数量、关卡、当前猪类型
3. 主菜单：开始游戏、关卡选择
4. 关卡选择：3关，已通关显示星级
5. 结算界面：星级+分数+下一关/重试/主菜单
6. 剩余猪队列显示（左下角）
7. 手机竖屏时显示"请横屏游玩"提示
8. 相机锁定/解锁按钮
