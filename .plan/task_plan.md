# Task Plan: 愤怒的猪 3D 代码重写

## 目标
按 PRD v2.1 完整重写游戏代码，走 engineering 流程：分块实现+逐块测试+code review。

## 当前状态
- PRD v2.1 已定稿（docs/generate_prd.py）
- 上一轮代码有基础但质量问题多：一次性写了1500行没逐步测试
- 已知 bug：HTML遗漏元素、连击倍率错误、鸟碰撞漏帧（已修但未系统验证）
- 现有代码在 GitHub Pages 可运行：https://zch133.github.io/angry-pigs/

## 任务拆分（薄切片，每块写完测完再下一块）

### Phase 1: 基础层
- [ ] T1: 常量定义+数值对齐 PRD（材料HP、分数、速度阈值、碰撞半径）
- [ ] T2: HTML 结构完整（所有 getElementById 引用都有对应元素）
- [ ] T3: CSS 样式（HUD布局、暂停菜单、飘字、连击文字、引导提示）

### Phase 2: 场景渲染
- [ ] T4: Three.js 场景初始化（相机、灯光、天空、地面、山丘）
- [ ] T5: 角色建模（猪3种、鸟2种、弹弓、建筑材料3种、碎片）
- [ ] T6: 鸟表情系统（正常/警觉/惊恐/被击中）

### Phase 3: 核心玩法
- [ ] T7: 弹弓拖拽+发射（射线投射到水平面、力度计算、松手发射）
- [ ] T8: 轨迹预测（物理模拟前0.5秒、15个点虚线）
- [ ] T9: 鸟碰撞检测（线段-球体相交，连续碰撞，防高速漏帧）
