# 愤怒的猪 3D — 代码重写计划

## 目标
完全重写代码，严格对齐 PRD v2.1，达到"能用、好玩、没bug"的发布标准。

## PRD v2.1 关键数值清单（必须精确对齐）

### 物理
- PIG_RADIUS=0.5, BIRD_RADIUS=0.4
- SLINGSHOT_POS={x:2, y:1.5, z:0}
- MAX_PULL=2.5, LAUNCH_POWER=16
- GROUND_Y=0
- 重力 -9.82

### 材料 (Issue 1)
- 冰: HP=1, 密度=0.5, 速度>5一击碎裂
- 木: HP=3, 密度=1.0, 每次撞击-1HP, 速度>8一击碎裂
- 石: HP=6, 密度=2.5, 每次撞击-1HP, 炸弹-3HP
- 分数: 冰500, 木1000, 石2000

### 炸弹猪 (Issue 10)
- 爆炸半径=2.5, 固定3点伤害, 线性衰减中心3→边缘1
- 对冰木: 一击碎(中心范围), 对石: 剩3HP
- 对鸟: 直接击落

### 飞速猪 (Issue 6)
- x2.5速度, 持续到撞击或落地, 不限时

### 分数 (Issue 2)
- 鸟5000/只, 精准命中+3000, 剩余猪10000/只
- 连击只算鸟, 0.8秒窗口, 倍率: 2连x1.5/3连x2/4连x3/5+连x5
- 连击倍率只作用于鸟分, 建筑物分不受影响

### 关卡
- 关卡1: 3普通猪, 1站鸟, 3木箱, 二星8000/三星12000
- 关卡2: 1飞速+2普通, 1飞鸟+1站鸟, 多层结构, 二星20000/三星30000
- 关卡3: 1普通+1飞速+1炸弹, 3鸟, 混合结构+山坡, 二星35000/三星50000

### 鸟 (Issue 5, 12)
- 飞鸟: 绿色#4CAF50, 圆心半径1.5unit, 慢速飘移
- 站鸟: 橙色#FF9800, 平台宽度1/2, 中速跑动
- 边界: AABB外扩0.3缓冲区, 动态更新
- 表情: 正常/警觉(3unit内)/惊恐(1unit内)/被击中

### 相机 (Issue 3)
- 默认斜俯视~35°
- 飞行中跟随, 停止后0.5秒停顿→1秒缓动回弹弓
- 仅装填/等待阶段可手动旋转, 瞄准/飞行/回程禁止
- 桌面: 右键拖拽旋转, 滚轮缩放
- 手机: 单指拖拽, 双指捏合

### 停止判定 (Issue 9)
- 速度<0.1持续0.5秒 = 回合结束
- 卡建筑: 淡出0.5秒, 物理体延迟0.5秒移除
- 飞出边界(x>50或x<-20) = 回合结束

### 物理预演算 (Issue 4)
- 关卡加载时1秒物理预演算(不渲染), 建筑settle后玩家可发射
