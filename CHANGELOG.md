# CHANGELOG

## v3.0.0 — 2026-07-17（正式发布）

### 范围扩大（对应 PRD v3.0）
- 关卡规模 3 关 → **100 关**（5 张主题地图 × 20 关），难度曲线递增
- 新增 5 套地图主题：青青草原 / 沙漠遗迹 / 雪山冰川 / 火山熔岩 / 云端天空城（专属配色、雾、远景装饰、材质侧重）
- 新增 **TNT 炸药箱**（1HP，受击爆炸，支持链式引爆），覆盖 43 关
- 第三方库全部本地化（three r128 + cannon 0.6.2 置于 `standalone/`），移除 CDN 依赖

### 手感与打击感（对齐经典弹弓玩法）
- 弹弓：橡皮筋双侧拉伸、力度条、14 点轨迹预测、拉伸音调随力度上升
- 相机：侧视角构图、按关卡宽度自动取景、发射平滑跟随、命中震屏（trauma 模型）、停球 0.5s 后 1s 缓动回位
- 打击反馈：击落慢动作（0.3× 550ms）、大冲击顿帧（60ms）、材质专属音效（冰脆/木闷/石沉）、碎片粒子、羽毛、飘分、连击横幅（×1.5/×2/×3/×5）
- 伤害模型：冰 1HP/木 3HP（裂痕两阶段）/石 6HP；速度阈值一击碎（冰>5、木>8）；爆炸中心 3→边缘 1 线性衰减
- 精准命中 +3000；剩余猪 +10000/只；三星阈值按关卡内容自动标定

### 工程与质量
- 修复 cannon 0.6.2 在碰撞回调中移除刚体导致崩溃的问题（统一延迟移除）
