# CHANGELOG

## v3.1.0 — 2026-07-17（试玩反馈迭代）

### 视野与镜头
- **装饰物自动渐隐**：挡在"相机→战场"视线走廊内的装饰（树木/岩石/仙人掌等）自动淡化至 12% 透明度，离开后平滑恢复；检测用包围盒中心（树冠高度）+ 点到线段距离，每 3 帧重算、逐帧 lerp 过渡
- 装饰摆放分区化：2/3 置于背景侧，1/3 置于前景两侧远处，从源头避免遮挡镜头走廊

### 手感调优（对齐原版弹弓节奏）
- **滞空更久**：飞行阶段重力系数 0.75×（仅离地生效）+ 轻微空气阻力，实测飞行时间 1.15s → 1.82s（+58%）
- **落地快停**：滚动阻力大幅增强（贴地低速强阻尼 + 角速度衰减 + 硬停阈值），实测落地滚动 1.88s → 0.92s（-51%）
- 发射功率 16 → 15.5，配合低重力保持射程一致性

### 瞄准与提示
- **拉弓锥形限制**：发射角钳制在 8°~78°（对齐原版有限角度手感），z 向偏移收窄 ±0.7，必须明显后拉（>0.15）才可发射
- **落点提示圈**：轨迹预测末端显示落点标记——预计撞建筑为红色圈、预计落地为黄色圈，带脉冲呼吸动画
- 无头仿真与求解器同步新物理模型（低重力 + 阻力 + 新滚动参数），100 关重建后三门禁（静态/物理/可通关）仍全量通过

### QA 增强
- harness 新增 `fadetest` 渐隐链路自检（移入走廊→opacity 0.12，移出→恢复 1.0）、日志显示渐隐计数
- `__game.ff()` 快进钩子覆盖渐隐更新（绕开后台 rAF 节流，QA 确定性复现）

### 工程
- 关卡数据分块化：`js/levels.js`（单文件 213KB）→ `js/maps.js` + `js/levels/chunk_0~5.js`（按地图分块，sky 图再拆两块以规避单文件体积上限，紧凑 JSON，总计 127KB，浏览器按序拼接 `window.LEVELS`）；verifier 经 `verifier/v1/load_levels.js` 聚合加载，三门禁重跑全过

## v3.0.0 — 2026-07-17（正式发布）

### 范围扩大（对应 PRD v3.0）
- 关卡规模 3 关 → **100 关**（5 张主题地图 × 20 关），难度曲线递增
- 新增 5 套地图主题：青青草原 / 沙漠遗迹 / 雪山冰川 / 火山熔岩 / 云端天空城（专属配色、雾、远景装饰、材质侧重）
- 新增 **TNT 炸药箱**（1HP，受击爆炸，支持链式引爆），覆盖 43 关
- 第三方库全部本地化（three r128 + cannon 0.6.2 置于 `standalone/`），移除 CDN 依赖

### 手感与打击感（对齐经典弹弓玩法）
- 弹弓：橡皮筋双侧拉伸、力度条、14 点轨迹预测、拉伸音调随力度上升
- 相机：侧视角构图、按关卡宽度自动取景、发射平滑跟随、命中震屏（trauma 模型）、停球 0.5s 后 1s 缓动回位
- 打击反馈：击落慢动作（0.3× 550ms）、大冲击顿帧（60ms）、材质专属音效（冰脆/木闷/石沉）、碎片粒子、羽毛、飘分、连击横幅（×1.5/×2/×3/×5）
- 伤害模型：冰 1HP/木 3HP（裂痕两阶段）/石 6HP；速度阈值一击碎（冰>5、木>8）；爆炸中心 3→边缘 1 线性衰减
- 精准命中 +3000；剩余猪 +10000/只；三星阈值按关卡内容自动标定

### 工程与质量
- 修复 cannon 0.6.2 在碰撞回调中移除刚体导致崩溃的问题（统一延迟移除）
- 新增滚动阻力（落地快速停球，对齐原版手感）
- 物理预演算期间伤害系统"不上膛"，避免结构沉降误引爆 TNT
- 关卡生产管线：生成器 + 稳定性验证 + 自动求解器 + 自动修复（加猪/换种子），100 关全量通过
- verifier 三门禁（静态/物理/可通关）+ 运行留档（`verifier/runs/`）
- 鸟 AI：飞鸟绕建筑飘移不穿插、站鸟平台跑动碰壁转身、警觉/惊恐表情、随机鸣叫
- 新手引导（第 1 关拖拽教学）、技能猪首次提示、竖屏提示、暂停菜单（音量调节）、关卡选择（5 地图分页 + 星级 + 锁定）
