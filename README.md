# 愤怒的猪 3D 🐷

一款 3D 物理解谜弹射网页游戏：小猪扛起弹弓，向霸占城堡的鸟儿发起反击。
经典弹弓抛射玩法 + 全 3D 场景 + **100 个关卡 · 5 大主题地图**。

**在线试玩**：https://zch133.github.io/angry-pigs/

## 玩法

- **拖拽**小猪向后拉 → 松手发射（有轨迹预测线与力度条）
- 全歼所有鸟即过关；用猪越少、砸得越狠，分数越高、星星越多（最高三星）
- 飞行中**点击屏幕**触发技能猪：
  - 💨 **飞速猪**：瞬间加速 2.5 倍，贯穿一切
  - 💣 **炸弹猪**：原地爆炸，冲击波掀翻建筑（中心伤害 3 → 边缘 1）
- 剩余猪每只 +10000 分；连续击落触发连击倍率（最高 ×5）

### 材质破坏规则

| 材质 | 耐久 | 说明 |
|------|------|------|
| 🧊 冰 | 1 | 一击即碎（速度 >5） |
| 🪵 木 | 3 | 速度 >8 可一击碎，逐击出现裂痕 |
| 🪨 石 | 6 | 最硬，爆炸一次 -3 |
| 🧨 TNT | 1 | 受击即爆，可连锁引爆 |

## 内容

- **5 张主题地图**：🌿 青青草原（教学）→ 🏜️ 沙漠遗迹（石头+山坡）→ 🏔️ 雪山冰川（冰面低摩擦）→ 🌋 火山熔岩（石头+TNT）→ ☁️ 云端天空城（混合高难度）
- 每图 20 关，共 **100 关**，难度曲线递增
- 关卡由生产管线生成（`tools/`），全部通过**无头物理稳定性验证**与**自动求解器可通关验证**

## 技术

- Three.js r128（`standalone/` 本地内置，无 CDN 依赖）
- cannon.js 0.6.2 物理引擎（本地内置）
- 全部美术（模型/贴图）程序化生成，全部音效 Web Audio 实时合成 —— **零外部素材**
- localStorage 本地存档（进度/星级/最高分）

## 本地运行

```bash
python3 -m http.server 8000
# 打开 http://localhost:8000/
```
