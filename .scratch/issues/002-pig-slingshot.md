## 实现猪的 3D 建模和弹弓发射

**From PRD:** /home/z/my-project/angry-pigs/PRD.md (R1, R2, R5, R10)
**Acceptance:**
- 三种猪的 3D 模型在场景中可见：普通猪（粉）、飞速猪（黄）、炸弹猪（黑）
- 每种猪有：圆滑球体身体、鼻子、耳朵、眼睛、尾巴，颜色区分种类
- 猪有简单待机动画（在弹弓上轻微晃动）
- 弹弓 3D 模型（Y形木叉）在场景左侧可见
- 拖拽发射功能正常：按住猪往后拖，显示弹弓拉伸效果 + 力度指示器
- 松手发射，猪按拖拽反方向飞出，3D 抛物线轨迹
- 拖拽 X 轴映射左右方向，Y 轴映射力度，自动带向上弧度
- 猪发射后受物理规则控制（重力、惯性）
- 猪落地后停止
- 飞行中点击屏幕触发特殊技能：飞速猪加速、炸弹猪爆炸
- 屏幕左下角显示剩余猪队列

### Context
- ADR-0001：cannon-es 物理引擎
- ADR-0002：代码精细建模
- 依赖 issue #001 的引擎骨架

### Scope
- In: 猪的 3D 建模、弹弓建模、拖拽发射控制、物理发射、特殊技能、剩余猪队列 UI
- Out: 鸟、建筑物、关卡设计、分数系统
