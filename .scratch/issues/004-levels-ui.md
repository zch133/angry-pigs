## 实现关卡系统、分数、存档和 UI

**From PRD:** /home/z/my-project/angry-pigs/PRD.md (R7, R8, R9, R12)
**Acceptance:**
- 3 关关卡数据定义，难度递增
- 第1关：教学关，1只鸟，简单结构，3只普通猪
- 第2关：进阶，2只鸟（飞鸟+站鸟），多层结构，含飞速猪
- 第3关：挑战，3只鸟，复杂结构，含炸弹猪和石块
- HUD 正常显示：分数、剩余猪、关卡、当前猪类型
- 主菜单：开始游戏、关卡选择
- 关卡选择界面：3关，已通关显示星级
- 结算界面：通关显示星级+分数+下一关/重试/主菜单；失败显示重试/主菜单
- 最后一关通关显示"恭喜全部通关"
- 分数计算：每击落一只鸟 +1000，每剩余一只猪 +500
- 星级：一星=通关，二星=达到 twoStar 线，三星=达到 threeStar 线
- localStorage 存档：最高解锁关卡、每关最高分、每关最高星级
- 刷新页面不丢进度

### Context
- 依赖 issue #002（猪系统）和 #003（鸟+建筑物）
- 关卡数据定义猪种类顺序、鸟位置和类型、建筑物布局、目标分数线

### Scope
- In: 关卡数据、HUD、主菜单、关卡选择、结算界面、分数计算、星级评定、存档系统
- Out: 引擎、猪/鸟/建筑物本身

### Notes
- 关卡数据用 JSON 风格定义，每关包含：pigTypes 数组、birds 数组、blocks 数组、twoStar/threeStar 分数线
- UI 用 HTML overlay 而非 3D 内 UI，更简单
