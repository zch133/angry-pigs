# 愤怒的猪 3D

愤怒的小鸟角色互换版——猪用弹弓发射自己去砸鸟。3D 立体版本，部署在 GitHub Pages，手机和电脑浏览器均可游玩。

## Language

**Pig（猪）**:
玩家发射的角色，用弹弓发射出去砸鸟。有三种类型。
_Avoid_: 玩家角色、发射体

**Bird（鸟）**:
游戏目标，需要被猪砸中击落。有飞鸟和站鸟两种状态。
_Avoid_: 目标、敌人

**Flying Bird（飞鸟）**:
在固定圆心范围内随机飘移飞行的鸟。飘移轨迹会避开建筑物，不穿过也不碰撞。
_Avoid_: 空中鸟

**Grounded Bird（站鸟）**:
在平台宽度 1/4 范围内左右来回跑动的鸟。跑到边缘转身，不会掉下平台。
_Avoid_: 地面鸟、跑鸟

**Block（建筑物/结构块）**:
可被猪破坏的物理刚体。有冰、木、石三种材质，耐久度不同。
_Avoid_: 障碍物、掩体

**Slingshot（弹弓）**:
发射猪的装置，固定在场景左侧。玩家拖拽猪往后拉，松手发射。
_Avoid_: 发射器

**Pig Type（猪种类）**:
普通猪（粉，无技能）、飞速猪（黄，点击加速）、炸弹猪（黑，点击爆炸）。
_Avoid_: 角色类型

**Material（材料）**:
冰（最脆，1次重击碎）、木（中等，2-3次重击碎裂）、石（最硬，需炸弹猪或多次撞击）。
_Avoid_: 材质

**Bird State（鸟状态）**:
鸟有两种状态——飞行/跑动状态（活着，与建筑物无碰撞但不穿过）和掉落状态（被击中后，受物理规则影响，与建筑物有碰撞）。
_Avoid_: 鸟的形态
