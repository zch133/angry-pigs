# Issues: 愤怒的猪 3D

> 对齐 PRD v2.1，按真实游戏开发流程拆分。每个 Issue 可独立验收。

---

## Issue 1: 材料与伤害模型对齐
**From PRD:** 7.1, 7.2, 7.3
**问题:** 材料HP值错误（15/40/100→1/3/6），伤害计算用速度×系数而非PRD的阈值模型，炸弹猪伤害用变量而非固定3点。
**Acceptance:**
- 冰 1HP，速度>5 一击碎裂
- 木 3HP，每次撞击-1HP，速度>8 一击碎裂
- 石 6HP，每次撞击-1HP，炸弹猪爆炸-3HP（线性衰减中心3→边缘1）
- 炸弹猪：固定3点伤害，不再用"force×2"公式
**Scope:**
- In: MATERIAL_HEALTH 常量、碰撞伤害计算、explode() 函数
- Out: 分数、连击

## Issue 2: 分数与连击系统
**From PRD:** 8.1, 8.2
**问题:** 分数值错误（1000/鸟→5000/鸟），无连击系统，无精准命中奖励，无剩余猪奖励。
**Acceptance:**
- 击落鸟 5000分/只，冰500/块，木1000/块，石2000/块
- 连击只计算鸟（不含建筑物），0.8秒窗口
- 连击倍率：2连x1.5 / 3连x2 / 4连x3 / 5+连x5
- 连击倍率只应用于鸟的得分，建筑物分不受影响
- 精准命中（直接击中鸟不经过建筑）额外3000分
- 剩余猪奖励 10000/只
**Scope:**
- In: score 计算、combo 状态机、精准判定
- Out: 飘字显示（Issue 7）

## Issue 3: 轨迹预测系统
**From PRD:** 5.3
**问题:** 拖拽时无预测轨迹线。
**Acceptance:**
- 拖拽弹弓时，用物理引擎模拟前0.5秒轨迹
- 显示虚线点（约10-15个点），松手后消失
- 预测准确，考虑重力和初始速度
**Scope:**
- In: 预测模拟、点渲染
- Out: 实际飞行轨迹

## Issue 4: 相机系统对齐
**From PRD:** 5.4
**问题:** 无飞行跟随，锁定时机错误（随时可解锁→仅装填/等待阶段可操作）。
**Acceptance:**
- 猪飞行中：相机平滑跟随猪，保持猪在画面中心偏上
- 猪停止后：0.5秒停顿 → 1秒缓动回到弹弓视角
- 相机手动操作仅限：装填阶段 + 等待阶段
- 瞄准拖拽开始时：相机自动切回弹弓视角并锁定
- 飞行中、回程动画中：禁止手动操作
**Scope:**
- In: camera follow 逻辑、camera lock 状态机
- Out: 角度限制（已有）

## Issue 5: 鸟系统对齐
**From PRD:** 6.2, 6.3, 6.4
**问题:** 鸟颜色错误，站鸟范围错误（1/4→1/2），无表情系统，飞鸟边界不够精确。
**Acceptance:**
- 飞鸟颜色：绿色 #4CAF50
- 站鸟颜色：橙色 #FF9800
- 站鸟活动范围：平台宽度的 1/2
- 飞鸟边界：圆心半径 + 建筑AABB外扩0.3unit缓冲区，动态更新
- 鸟表情：正常/警觉（猪3unit内）/惊恐（猪1unit内）/被击中
**Scope:**
- In: bird颜色、runRange、表情动画、边界检测
- Out: 鸟建模（已有）

## Issue 6: 猪技能与状态对齐
**From PRD:** 5.2, 6.1, 6.5
**问题:** 飞速猪加速是瞬间而非持续，炸弹猪伤害模型不对，猪卡建筑处理突变，停止判定阈值不对。
**Acceptance:**
- 飞速猪：点击后速度x2.5，持续加速直到撞击或落地，方向不变
- 炸弹猪：固定3点伤害，线性衰减（中心3→边缘1）
- 猪卡建筑：0.5秒淡出动画 + 物理体延迟0.5秒移除
- 停止判定：速度<0.1 持续0.5秒
- 出界判定：飞出关卡边界
**Scope:**
- In: speed boost 逻辑、explode 伤害、pig stuck 处理、stop 检测
- Out: 材料HP（Issue 1）

## Issue 7: 反馈与引导系统
**From PRD:** 9.1, 9.2, 13.3
**问题:** 无飘字分数，无连击飘字，无技能引导提示，无新手引导。
**Acceptance:**
- 破坏建筑物：飘字 "+500" 等，在破坏位置浮起淡出
- 击中鸟：飘字 "+5000"
- 连击触发：屏幕中央大字 "x1.5" 等，0.8秒淡出
- 首次使用飞速猪/炸弹猪：猪身闪烁光效 + 底部"点击触发技能"提示
- 第1关首次进入：弹弓旁有动画手指引导拖拽
**Scope:**
- In: floating text 系统、combo text、skill hint、newbie guide
- Out: 音效（已有）

## Issue 8: 关卡与流程对齐
**From PRD:** 10.1, 10.2, 10.3, 12.1
**问题:** 无物理预演算，第3关无山坡，无暂停菜单，关卡数据不完整。
**Acceptance:**
- 关卡加载时1秒物理预演算（不渲染），建筑稳定后玩家才可发射
- 第3关有山坡（静态物理碰撞体），猪可撞击被弹开
- 暂停菜单：继续/重新开始/返回主菜单 + 音量滑块
- HUD：左上角关卡+暂停，右上角分数+相机，左下角猪队列
**Scope:**
- In: physics pre-sim、level 3 hill、pause menu、HUD layout
- Out: 核心玩法

## Issue 9: 音效对齐
**From PRD:** 11.1, 11.2
**问题:** 缺少弹弓拉伸声、连击音效。
**Acceptance:**
- 拖拽弹弓时播放拉伸声（音调随力度上升）
- 连击触发时播放对应音效（叮/上升音阶/激昂/震撼）
- 现有音效保留
**Scope:**
- In: stretch sound with pitch、combo sounds
- Out: 现有音效
